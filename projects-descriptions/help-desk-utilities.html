Help Desk Utilities

Description:

I built this website in order to collect useful information for our Help Desk technicians to use in order to solve problems quicker and more efficiently. Some utilities include an "Email Scrubber," a MAC Finder Java applet, list of links to common anti-virus used and useful software.

Some of the Help Desk technicians are also programmers and we have had a problem with email submission form where we get sent raw HTML instead of a rendered email. We had tried to use C++ to crawl the HTML and pull out the contents of the email. It wasn't hard to put together, but I found a simpler solution using PHP to pull the email into a string and echoing it out, which will allow the browser to render it normally.

Components:

  • CSS
  • Twitter Boostrap
  • PHP
