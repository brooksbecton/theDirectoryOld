Soybean Road Quiz

Description:

I was recommended to a professor to create a quiz type web page to test younger students on their knowledge of road safety in the context of farming equipment.

The site strictly uses JavaScript and jQuery to change the content of the page. To store the results, I am using JavaScript to package the results of the quiz into a JSON object and using a quick PHP write to a file to store the results.

As far as the professor's wants are concerned, I am finished with this project, but their are a few responsiveness quirks that I would like to fix and provide to the client
