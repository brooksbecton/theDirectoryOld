Meme Generator

Description:

This project started as a project for my Web Development class. The goal of the project was to use php to edit text over a background image to create memes. One of the issues was that you could not download the meme to share. I decided to remake the project using HTML5 Canvas. You are now able to download the meme you create.

Components:

  • HTML
  • CSS
  • Twitter Boostrap
  • JavaScript
  • jQuery
  • PHP
