Sigma Alpha Test Site

Description:

I created this page to represent the Sigma Alpha Sorority Beta Theta chapter at the University of Tennessee at Martin because I found that their previous was out of date.

I mainly would like to make this site editable with WYSIWYG and CMS platform so that the members will be able to edit their pages without requiring external help while also maintaining a level of security to prevent XSS and other types of security exploits

I am also using Angular JS specifically for the SPA aspects to more efficiently serve pages out to viewers on the website. I should also be able to to use the controllers to better serve pages quicker.

Components:

  • HTML
  • CSS
  • Twitter Bootstrap
  • Angular JS
