Breakout

Description:

This is a simple breakout game using HTML5 Canvas. It was very interesting to see how you could combine the canvas in JavaScript to create this iconic game. This was first experience when any kind of game logic.

Understanding redrawing the canvas and refresh rates was very interesting considering how much modern video game players take this logic for granted.

I have started to research Node JS and hope to combine the two for this project in order to provide on line capabilities for users.

Components:

  • HTML5 Canvas
  • JavaScript
